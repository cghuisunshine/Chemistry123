Chapter 24: The Chemistry of Organic Compounds

Learning Objectives

By the end of this tutorial, you should be able to:

  • Recognize common classes of organic compounds (alkanes, alkenes, alkynes, alcohols, carbonyls, carboxylic acids, etc.).
  • Draw and interpret condensed, line‑angle, and structural formulas.
  • Apply basic IUPAC naming rules for simple organic molecules.

Key Vocabulary

  • Organic compound: carbon‑based compound, often containing C–H bonds.
  • Functional group: specific group of atoms that largely determines reactivity (for example, OH, C=O, CO2H).
  • Alkane: hydrocarbon with only single bonds (CnH2n+2).
  • Alkene: hydrocarbon with one or more C=C double bonds.
  • Alkyne: hydrocarbon with one or more C≡C triple bonds.
  • Isomer: compounds with the same molecular formula but different structures.

1. Representations and Functional Groups

  • Line‑angle formulas omit C and H explicitly; each vertex or line end is a carbon with implied hydrogens.
  • Recognize key functional groups: alcohols (OH), aldehydes and ketones (C=O), carboxylic acids (CO2H), amines (NR3).

2. Simple Naming Ideas

  • Base name from longest carbon chain: meth‑, eth‑, prop‑, but‑, pent‑, …
  • Suffix indicates functional group: ‑ane, ‑ene, ‑yne, ‑ol, ‑al, ‑one, ‑oic acid.
  • Number chain to give lowest possible locants for key features.

3. Practice Problems (Self-Check)

  1. Identify the functional groups present in: CH3CH2OH, CH3COCH3, and CH3CH2CO2H.
  2. Draw line‑angle formulas for propane, 2‑butanol, and propanone (acetone).
  3. Give reasonable IUPAC names for the following (sketch if needed): (a) CH3CH2CH2CH3; (b) CH3CH(CH3)CH3; (c) CH3CH2CH2OH.
