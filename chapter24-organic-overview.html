Chapter 24: The Chemistry of Organic Compounds

Learning Objectives

By the end of this tutorial, you should be able to:

  • Recognize common classes of organic compounds (alkanes, alkenes, alkynes, alcohols, carbonyls, carboxylic acids, amines).
  • Draw and interpret condensed, line‑angle, and full structural formulas.
  • Apply basic IUPAC naming rules for simple organic molecules.

Key Vocabulary

  • Organic compound: carbon‑based compound, often containing C–H bonds.
  • Functional group: specific group of atoms that largely determines reactivity (for example, OH, C=O, CO2H, NH2).
  • Alkane: hydrocarbon with only single C–C bonds (CnH2n+2).
  • Alkene: hydrocarbon with C=C double bond(s).
  • Alkyne: hydrocarbon with C≡C triple bond(s).
  • Isomer: compounds with the same molecular formula but different connectivity or spatial arrangement.

1. Representations and Functional Groups

  • Line‑angle formulas omit C and H explicitly; each vertex or line end is a carbon with implied hydrogens.
  • Recognize key functional groups:
    • Alcohols (OH), ethers (C–O–C).
    • Aldehydes and ketones (C=O), carboxylic acids (CO2H).
    • Amines (NR3), amides (CONR2).

2. Simple Naming Ideas

  • Base name from longest carbon chain: meth‑, eth‑, prop‑, but‑, pent‑, hex‑, etc.
  • Suffix indicates main functional group: ‑ane (alkane), ‑ene (alkene), ‑yne (alkyne), ‑ol (alcohol), ‑al (aldehyde), ‑one (ketone), ‑oic acid (carboxylic acid).
  • Number chain to give lowest possible locants for multiple bonds and substituents.

Worked Example – Naming a Simple Alcohol

Give a reasonable IUPAC name for CH3CH2CH2OH.

  1. Longest chain: 3 carbons → base name prop‑.
  2. Functional group: OH (alcohol) → suffix ‑ol.
  3. Number chain so OH gets lowest number (1) → 1‑propanol (often called propan‑1‑ol).

3. Practice Problems (Self-Check)

  1. Identify the functional groups present in: CH3CH2OH, CH3COCH3, CH3CH2CO2H.
  2. Draw line‑angle formulas for propane, 2‑butanol, and propanone (acetone).
  3. Give reasonable IUPAC names for: (a) CH3CH2CH2CH3; (b) CH3CH(CH3)CH3; (c) CH3CH2CH2OH.
  4. Describe in words or sketches the difference between an isomer pair, such as n‑butane and isobutane.
